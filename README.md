# YouTube Clone

## Project Overview

This project is a YouTube Homepage Clone created using HTML, CSS, and JavaScript. The website recreates the basic layout and design of YouTube, including a navigation bar, sidebar, category buttons, search functionality, and responsive video cards.

## Features

* Responsive Design
* Navigation Bar
* Sidebar Menu
* Search Functionality
* Video Grid Layout
* Hover Effects
* Mobile Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Files Included

* index.html
* style.css
* script.js
* README.md

## How to Run

1. Open the project folder.
2. Open index.html in your browser.
3. Explore the YouTube Clone interface.
